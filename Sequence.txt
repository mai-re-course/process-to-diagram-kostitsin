Пользователь -> Браузер : Ввод запроса
Браузер -> Браузер : Парсинг URL
Браузер -> Браузер : Поиск соответствия в списке HSTS

alt Сайт есть в списке HSTS
    Браузер -> Браузер : Выбор протокола HTTPS
else Сайта нет в списке HSTS
    Браузер -> Браузер : Выбор протокола HTTP
end

Браузер -> Браузер : Поиск адреса в DNS-кэше

alt Адреса нет в DNS-кэше
    Браузер -> DNS_сервер : ARP-запрос
    DNS_сервер -> Браузер : ARP-ответ, извлечение IP
else Адрес есть в DNS-кэше
    Браузер -> Браузер : Извлечение IP из кэша
end

Браузер -> Браузер : Запрос потока TCP-сокета
Браузер -> Сервер : TLS handshake (ClientHello)
Сервер -> Браузер : Ответ на TLS handshake (ServerHello)
Браузер -> Сервер : Client Finished
Сервер -> Браузер : Server Finished
Браузер -> Сервер : Отправка серверу HTTP-запроса

alt Ответ сервера 
    Сервер -> Браузер : Ответ 200 ОК
else Файл не менялся с последнего запроса
    Сервер -> Браузер : Ответ 304 Not Modified
    Браузер -> Браузер : Получение HTML из кэша
end

Браузер -> Браузер : Парсинг HTML
Браузер -> Браузер : Интерпретация CSS
Браузер -> Браузер : Рендеринг страницы
Браузер -> Пользователь : Отрисованная страница
